# PawMap

PawMap is a web application designed to help pet owners manage their pets' health care needs and locate nearby veterinary services. The application provides an intuitive interface for tracking pet information, medications, and finding veterinary care.

## Features

- **Pet Management**
  - Add and manage pet profiles
  - Track pet medications and schedules
  - View pets in grid or list layout
  - Store important pet information and medical history

- **Veterinary Services**
  - Find nearby veterinary clinics and hospitals
  - Access emergency contact information
  - Schedule appointments
  - AI-powered veterinary assistant

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/pawmap-ui.git
cd pawmap-ui
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

The application will be available at `http://localhost:3000`
